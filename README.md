# Great Market V1 - Portfolio

Portfolio de Great Market, une application React moderne construite avec Vite.

## 🚀 Technologies

- React 18
- Vite 8
- CSS3

## 📦 Installation

```bash
npm install
```

## 🛠️ Développement

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
